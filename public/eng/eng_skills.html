"Meteorite Destroyers"

My first full stack application. I developed this game originally as a project for a full stack programming course in Fall of my second year. As a freetime hobby I have continued to develop it by adding improvements and features every now and then. In the game you play as a ship and the objective is to destroy as many meteorites as possible. The player has 5 lives and once they run out the game ends. If the player has reached a personal record highscore it is saved to a database. Technologies used: JavaScript, Phaser, NodeJS, Express, RestAPI, MongoDB Full Stack -project (Fall 2023) GitHub repo
