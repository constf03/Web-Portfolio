Website for restaurant Ravintola Johnny Hong Kong Hyvink채채

Internship Summer 2024 - Tmi J.J.Wessman

Topic: full stack development, cloud services, continuous integration (CI)
During my internship I created a website for restaurant Ravintola Johnny Hong Kong Hyvink채채 (link, note 4.11.2024: could be temporarily unavailable due to a cloud platform migration). I implemented the following features:
  • user authentication
  • user data updating
  • products to food menu (JSON structure)
  • admin user feature
  • a simple food-ordering system
  • localization in Finnish and English
  • responsive UI
The site was first hosted on AWS' Elastic Container Service where frontend and backend Docker containers ran on https protocol, but it was later moved to another platform. Code was deployed through Codebuild on AWS.
Main technologies used: Node.js, Express, React, TypeScript, RestAPI, PostgreSQL, Docker Other (frameworks, libraries etc.): Sequelize ORM, TailwindCSS, Vite, Auth0 Service platforms: GitHub, Amazon Web Services (Elastic Container Service, Codebuild, Route 53), Aiven
